# Portafolio de Fernando Navas — GitHub Pages (EJS → HTML estático)


## Requisitos
- Node.js 18+


## Desarrollo local
```bash
npm install
npm run dev
# abre http://localhost:3000 (servidor estático para /docs)
```


## Build
```bash
npm run build
# genera HTML estático en /docs (index.html, projects.html, experience.html, contact.html)
