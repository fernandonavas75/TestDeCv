Mini tienda para colegio

Este proyecto es una mini tienda en línea desarrollada para un colegio, que permite a los estudiantes y personal comprar productos escolares y otros artículos directamente desde la plataforma. La aplicación está construida utilizando Node.js y Express para el backend, con EJS como motor de plantillas para renderizar las vistas del lado del servidor. La tienda ofrece una interfaz amigable donde los usuarios pueden navegar por los productos, agregarlos al carrito de compras y proceder al pago. Además, se implementaron funcionalidades de administración para que el personal del colegio pueda gestionar el inventario de productos, ver órdenes y actualizar detalles de los artículos disponibles.
